# 社区康养中心系统 (Community Wellness Center)

## 1. 项目简介
本项目是一个面向社区老人的**智慧康养服务平台**。旨在通过简单的数字化手段，为老年人提供便捷的健康管理、生活照护、商城购物及安全监护服务。
- **核心用户**：社区内的老年人（会员端）、服务机构工作人员（机构端，规划中）、平台管理员（平台端，规划中）。
- **解决痛点**：解决老年人操作智能设备困难、获取服务不便、紧急情况求助难的问题。

## 2. 系统整体架构
目前处于**第一阶段（原型开发）**，主要侧重于前端展示和交互体验。
- **技术栈**：
  - **后端/服务器**：Node.js + Express (目前仅作为静态资源服务器)
  - **前端**：原生 HTML5 + Tailwind CSS (无需编译，轻量级)
  - **图标**：FontAwesome
- **数据流**：目前使用静态模拟数据，暂未连接数据库。

## 3. 页面结构说明 (会员端 - 移动版)
会员端专为老年人设计，强调**大字体、高对比度、操作简单**。

### 3.1 登录页 (`/member/login.html`)
- **用途**：用户身份验证。
- **特点**：超大输入框，仅需手机号和验证码，减少记忆负担。
- **跳转**：验证成功后跳转至首页。

### 3.2 首页 (`/member/index.html`)
- **用途**：功能的聚合入口。
- **布局**：
  - **顶部**：个人问候、天气、步数（情感化设计）。
  - **功能网格**：健康档案、照护服务、生活商城、安全设备。
  - **SOS 紧急呼叫**：穿插在功能区中间，醒目且易触达。
  - **通知区**：社区公告和每日提醒。
- **底部导航**：固定在底部，方便在主要功能间快速切换。

### 3.3 健康档案 (`/member/health.html`)
- **用途**：展示老人的健康数据。
- **功能**：
  - 个人基本信息卡片。
  - 血压/血糖大字展示。
  - 血压趋势折线图。
  - 历次评估报告列表。

### 3.4 照护计划 (`/member/care.html`)
- **用途**：查看每日的服务安排。
- **功能**：
  - 周历选择器。
  - 今日服务清单（已完成/待完成状态区分）。
  - 历史服务记录。

### 3.5 商城订单 (`/member/orders.html`)
- **用途**：生活物资和服务的采购记录。
- **功能**：
  - 订单状态分类（配送中/已完成）。
  - 商品大图展示。
  - 操作按钮（确认收货/再次购买）。

### 3.6 设备与安全 (`/member/device.html`)
- **用途**：紧急求助和设备管理。
- **功能**：
  - **超大 SOS 按钮**：带脉冲动画，一键呼救。
  - 紧急联系人列表（支持直接拨号）。
  - 智能设备状态监控（手环、报警器）。

## 4. 如何运行
1. **安装依赖**：
   ```bash
   npm install
   ```
2. **启动服务器**：
   ```bash
   node server.js
   ```
3. **访问**：
   打开浏览器访问 `http://localhost:3000` 即可进入会员端登录页。
